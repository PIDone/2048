How to play (scroll down to read)

This is an implementation of the game 2048.

Use the Arrow Keys, WASD keys or the IJKL keys to shift the tiles up, left, down or right, respectively.

If two identical tiles are next to each other, shift in that direction and they will merge.

Every time you merge two tiles, the score will increase by the number on the resulting tile.

For example, if there are two 8 tiles next to each other horizontally, shifting left or right will merge them into a 16 tile. The score also increases by 16.

Every time you make a move, a new tile will generate on a random empty tile. It will be either 2 or 4.

If the board is full and there are no tiles to merge, the game ends.


The default board size is 4, but you can change it with the "resize board" button.

You are also able to resize the displayed size of the board with the "resize board" button.


I have also written an algorithm to play the game. Press the "Start AI" button to start it.
